# Tarefas e Melhorias de Design (novo-membro.html)

- [ ] **Seções Retráteis (Acordeões):** O formulário é bem longo. Podemos fazer com que os blocos ("Endereço", "Dados Complementares", etc.) possam ser "encolhidos" ou "expandidos" clicando no título. Isso deixa a tela muito mais limpa e focada.
- [ ] **Ícones nos Títulos das Seções:** Podemos adicionar os ícones da biblioteca Phosphor nos títulos. Ex: Um ícone de usuário ph-user em Dados Pessoais, um de mapa ph-map-pin no Endereço. Ajuda muito na navegação visual.
- [ ] **Máscaras Automáticas (Auto-formatação):** Aplicar um script visual para que, ao digitar o CPF, o CEP ou o Celular, os tracinhos e parênteses sejam colocados automaticamente no input enquanto a pessoa digita.
- [ ] **Campos com Ícones Internos:** Colocar pequenos ícones dentro das caixas de texto. Por exemplo, o símbolo do @ já fixado sutilmente dentro da caixa do Instagram.
